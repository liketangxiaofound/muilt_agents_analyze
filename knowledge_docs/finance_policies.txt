财务管理制度（节选）

一、差旅费报销标准
1. 国内出差：普通人员住宿标准为每人每天不超过350元，部门负责人不超过500元。
2. 交通费：高铁二等座、飞机经济舱。特殊情况需乘坐一等座或商务舱的，须提前报批。
3. 伙食补助：每人每天100元，按实际出差天数计算。
4. 市内交通：每人每天80元包干使用，不需提供票据。

二、办公用品采购
1. 单项采购金额在500元以下的，由科室负责人审批。
2. 单项采购金额在500-2000元的，须报分管领导审批。
3. 单项采购金额超过2000元的，须经领导班子集体研究决定。
4. 所有采购须在政府采购平台下单，不得私自采购。

三、项目资金管理
1. 专项资金必须专款专用，不得挪作他用。
2. 项目预算执行率低于80%的，下年度预算将相应核减。
3. 项目结余资金原则上收回统筹，特殊情况可申请结转，但不得超过原预算的30%。
4. 跨年度项目需每年提交执行进度报告。

四、报销审批流程
1. 经办人填写报销单 → 科室负责人审核 → 财务科复核 → 分管领导审批 → 出纳付款。
2. 金额超过10000元的，还需单位主要负责人审批。
3. 所有报销须附原始票据，电子发票需附验真截图。
4. 报销时限：业务发生后30日内完成报销，逾期不予受理。
